## üìû Emergency Playbook

### If engagement drops:
1. Check if algorithm changed (search Twitter/LinkedIn)
2. Review last 5 posts - what's different?
3. Test a different content format
4. Increase comment activity on others' posts

### If connection accepts drop:
1. Review your profile - anything changed?
2. Check if you're being too salesy in requests
3. Try sending requests with no note
4. Verify you're targeting the right personas

### If responses drop:
1. Review message templates - too long? Too salesy?
2. Check timing of messages
3. Add more personalization
4. Test different CTAs

### If meetings aren't converting:
1. Review qualification criteria
2. Check if you're reaching decision makers
3. Improve pre-call materials
4. Follow up more persistently
