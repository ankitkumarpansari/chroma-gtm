## Best Practices

### Do's ‚úÖ
- Personalize the first line
- Keep messages under 100 words
- Include a clear but soft CTA
- Reference something specific about them
- Follow up (but not more than 4 times)

### Don'ts ‚ùå
- Don't pitch in the connection request
- Don't send walls of text
- Don't be pushy or salesy
- Don't use generic templates verbatim
- Don't follow up daily

### Timing Tips
- **Best days:** Tuesday, Wednesday, Thursday
- **Best times:** 8-10 AM, 12-1 PM (their timezone)
- **Avoid:** Monday mornings, Friday afternoons, weekends
- **Response window:** Reply within 24 hours

## A/B Testing Framework

### Test These Variables:
1. **Subject lines** (for InMails)
2. **Opening lines**
3. **CTA phrasing**
4. **Message length**
5. **With/without links**

### Track These Metrics:
| Metric | Target |
|--------|--------|
| Connection accept rate | 30%+ |
| Response rate to first message | 20%+ |
| Meeting book rate | 10%+ |

### Sample A/B Test:

**Version A:**
```
Hi [Name], saw we're both building in the AI space. Would love to connect!
```

**Version B:**
```
Hi [Name], been following [Company]'s work on [specific]. Would love to connect!
```

Track accept rates for 50 sends each, then use the winner.
