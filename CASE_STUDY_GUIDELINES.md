### Technical Depth Guidelines

For Cohort 1 (technical evaluators), include:

1. **Specific Chroma features used**
   - Semantic search
   - Regex/exact match
   - Package Search MCP
   - Multi-tenancy
   - Filtering

2. **Architecture context**
   - How Chroma fits in their stack
   - Integration points
   - Data flow

3. **Scale indicators**
   - Number of vectors/embeddings
   - Query volume
   - Data size
